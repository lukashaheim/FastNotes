# FastNotes

This project contains two versions of the FastNotes app:

- React Native version
- Flutter version

Both apps allow you to create and view notes.

---

# Requirements

Install the following tools:

- Node.js → https://nodejs.org
- Android Studio (for emulator)

Optional:
- Expo Go (for running the React Native app on your phone)

---

# Run React Native App

Go to the React Native folder:

Install packages:

npm install

Start the app:

npx expo start

Scan the QR code with Expo Go on your phone  
or run it in the Android emulator.

---

# Run Flutter App

Go to the Flutter folder:

Install packages:

flutter pub get

Run the app:

flutter run
